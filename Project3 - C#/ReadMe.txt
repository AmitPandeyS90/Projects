Salient features of the application

1. All data is dynamically generated based on the information from API
2. Used TableLayout (Not covered in class) in Resources and News tab
3. Used Flowlayout in Undergraduate, Graduate, Research By Interest area and faculty.
4. Buttons in Undergraduate and Graduate tabs pick colors randomly from an array. Color changes everytime the solution is run.
5. Used drag and drop feature in Research By Faculty (see left on Research Tab)
6. Application has an icon
